The Language of Algebra

The language of algebra involves using symbols, numbers and letters. The symbols stand for mathematical operations to perform, such as multiplication; the letters represent unknown quantities called variables. A common theme in algebra is to execute operations with the quantities given in a problem in order to evaluate an expression or find a numerical solution for a variable in an equation.

For example: "seven decreased by the product of w and the square of the sum of x and y can be written in the language of algebra as:

7-w[(x+y)2]

You can simplify this expression if values are substituted for all the variables.

Knowledge of this language will be helpful in many situations in our everyday lives, such as calculating how much money to leave for a tip in a restaurant. When we only have part of the information, we can use algebra to solve the unknown part. With algebra, we can solve such practical problems readily.

In order to evaluate expressions, a certain "order" must be followed so that we can all arrive at the same answer. Take a look at the two ways in which this expression was evaluated.

RIGHT WAY WRONG WAY
3 x 22 - 4 3 x 22 - 4
3 x 4 - 4 62 - 4
12 - 4 36 - 4
8 32

How can we arrive at two different answers? Although both appear to be mathematically correct, the order in which the operations were performed is different. The correct answer is 8. Certain rules of order must be followed to arrive at the one correct answer.

EXAMPLE 3

x3 - y2

Evaluate the above expression when x = 2 and y = — 4.

First, plug in each value of the variable making sure to use parentheses around the number:
(2)3 — ( — 4)2.

Next, simplify the expression according to the correct order of operations. Since there is nothing to simplify within the parentheses, we will move on to exponents. 23 = 8 and ( — 4)2 = 16.

Now we have the expression 8 — 16, which equals —8.

Evaluate the above expression when x = —2 and y=3

Like before, let's substitute each value of the variable into the expression making sure to use parentheses around each value ( — 2)3 — 32

Next, simplify the expression according to the correct order of operations. Again, since there is nothing to simplify within the parentheses, we will move on to the exponents.
( — 2)3 = -8 and 32 = 9

Now we have the expression —8 — 9, which equals — 17.

Take Note: —x2 ≠ (— x)2

You must remember that when simplifying exponents, the number that immediately precedes the exponent is getting raised to that power. For instance,
-32 = - 9.
Why is that? Because the 3 is the base that immediately precedes the exponent. You must square the 3 first and then multiply by the negative sign in front. 		On the contrary,
( — 3)2 = 9.
Why is that? Because the -3 is the entire base because it is in parentheses, which are being raised to the second power. Therefore, we must square the entire -3 resulting in -3 times -3, which is positive 9.
